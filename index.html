Known issues:

  • Doesn't work in Internet Explorer <10 due to CORS limitations.
  • Performance will be impacted in older browsers that don't support local storage.
  • Uses generic icons for categories of items since there's no appearance data in API.
  • Chat link tags will appear only in certain browsers (dependent on window.btoa).
  • Some weapons and armor are missing attribute bonuses (API deficiency). Seems to mostly affect named exotics from PvP sets.
  • No search/sort. (Partly an API deficiency but could be mitigated if I chose to manipulate data locally.)
